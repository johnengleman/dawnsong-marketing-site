# Space gallery paintings

Painted scenes for the landing-page gallery. The gallery loads
`/spaces/<slug>-light.webp` and `/spaces/<slug>-dark.webp` for each space. If a
file is missing, that tile falls back to a placeholder grid.

- Format: **.webp** (copied from the app's `assets/demo-space-art`, with the
  `-v2` version suffix stripped). Landscape, ~3:2. No text or UI baked in; the
  space name is overlaid by the site.
- The top gallery row uses the **light** versions, the bottom row uses the
  **dark** versions, regardless of the visitor's theme, so both palettes always
  show.
